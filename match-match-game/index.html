Match Game

Train your memory with a memory game. If you like puzzles or other quizzes, this memory game is for you.

The playing field consists of cards with different patterns on the back. Before the start of the game session, the player is shown the location of all paired cards, after 30 seconds they are hidden. The victory is counted when all pairs of cards are found.
